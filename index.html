git config user.name
Configura um nome de usuario

git config user.email
Configura um email de usuario

git init
Prepara uma pasta para o git

git status
Exibe a situção do arquivos no repositório, se os arquivos foram ou não adicionados para a area de staging

git add
adiciona os arquivos para a area de staging, que é a area prepada para o commit

git commit
Envia os arquivos da area de staging para o repositório local

git commit
Envia os arquivos da area de staging para o repositório local

git log
exibe todos os commits feitos pelos os usuarios, com datas, nome de usuario e email

git branch
Cria uma nova ramificação para o armazenamento dos arquivos, para alteração sem afetar o projeto funcional

git checkout
permite trocar a branch atual

git merge
envia as alterações de uma branch para outra branch, mantendo a atualização final das duas, juntando assim suas informçãos em uma unica branch

git pull
Importa os commits do repositorio remoto para o repositório local

git push
exporta os commits para o repositório remoto

git clone
Cria a cópia de um repositorio
